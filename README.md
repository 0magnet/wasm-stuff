# wasm-stuff

Chaotic Attractors & 3D models in Golang WebAssembly (WASM) Example

## Run

```
go run github.com/0magnet/wasm-stuff@master
```

Access the interface at:
* [127.0.0.1:8080/](http://127.0.0.1:8080/)
* [127.0.0.1:8080/tinygo](http://127.0.0.1:8080/tinygo)

OR:
* [0magnet.github.io/wasm-stuff/](https://0magnet.github.io/wasm-stuff/)
* [0magnet.github.io/wasm-stuff/tinygo](https://0magnet.github.io/wasm-stuff/tinygo)

### Serverless

The wasm is included within the html document itself, base64 encoded:
* [index.html](index.html)
* [tinygo/index.html](tinygo/index.html)
